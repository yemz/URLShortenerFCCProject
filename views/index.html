URL Shortener Microservice

(Back End FreeCodeCamp Project)

Tools: AWS - EC2 (Ubuntu), Apache, MongoDB (mLab), Node.js, Glitch

User stories:

    1. I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
    2. If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
    3. When I visit that shortened URL, it will redirect me to my original link.

Original URL Test:

https://urlshortener-fccproject.glitch.me/new/http://yahoo.com

Output:

{ original_url: "http://yahoo.com", short_url: "https://urlshortener-fccproject.glitch.me/HyakfOZQz" }

Short URL Test:

https://urlshortener-fccproject.glitch.me/HyakfOZQz

Redirect to Original URL:

http://yahoo.com
